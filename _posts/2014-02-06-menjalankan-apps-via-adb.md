Caranya mudah, silahkan ikuti langkah-langkah dibawah ini.

1. buka adb shell

   ```bash 
   add shell
   ```

2. lalu jalankan perintah am start + aplikasi yang ingin kita jalankan

   ```bash
   am start com.cyanogenmod.filemanager
   ```
